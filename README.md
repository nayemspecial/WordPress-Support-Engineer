A complete learning repository for becoming a **WordPress Support Engineer**, combining **Web Design** and **Web Development** with structured course outlines and YouTube video playlists.

This repository is designed for students and beginners who want to build a strong foundation in:
- Frontend (HTML, CSS, Tailwind CSS)
- Backend (PHP, MySQL)
- WordPress Theme & Widget Development

All lessons are supported with:
- Course outlines  
- Practical examples  
- YouTube video tutorials
